## Introduction

Stitching is where bag construction becomes precision engineering. The difference between a bag that lasts 20 uses and one that lasts 100 often comes down to thread choice, stitch density, and seam type—details that cost pennies but save dollars in customer satisfaction and returns. This guide provides the exact specifications you need to ensure your sewn nonwoven bags meet professional standards.

You’ll learn:

- Which thread types and sizes are standard for nonwoven construction
- How stitches per inch (SPI) affects strength and appearance
- The differences between overlock, chain stitch, and lock stitch
- Handle attachment methods from box X to cross stitch
- Quality checkpoints to catch defects before shipment

## Thread Types & Sizes

### Polyester 40/2, 20/2 Common Specs

Thread selection balances strength, cost, and sewing performance:

| Thread Spec | Denier | Breaking Strength | Typical Use | Cost Index |
|------------|--------|-------------------|-------------|------------|
| Polyester 60/2 | Light | 800–1,000 g | Lightweight bags, overlocking | 1.0× |
| Polyester 40/2 | Medium | 1,200–1,500 g | Standard bags, general seams | 1.2× |
| Polyester 20/2 | Heavy | 2,000–2,500 g | Heavy-duty bags, handles | 1.5× |
| Polyester core-spun 30/2 | Medium-heavy | 1,800–2,200 g | High-speed sewing, smooth finish | 1.6× |
| Bonded nylon 69 (T-70) | Extra heavy | 3,000–3,500 g | Industrial bags, maximum strength | 2.5× |

**Thread size notation explained**:
- First number (40, 20): Yarn count—lower numbers are thicker
- Second number (/2): Number of plies twisted together
- Metric alternative: Tex (grams per 1,000 meters); Tex 40 ≈ 40/2

### Thread Color Matching
- **Exact match**: Thread dyed to Pantone specification
- **Tone-on-tone**: Thread slightly darker than fabric (hides dirt, looks cleaner)
- **Contrast**: Deliberate color difference for decorative effect

**Best practice**: For invisible seams, match thread to fabric color. For decorative or reinforced seams, use contrast thread to signal quality construction.

## Stitch Density (SPI)

### 7–9 SPI Standard, 10+ SPI Heavy Duty

**Stitches Per Inch (SPI)**—also called stitch pitch—is the primary variable controlling seam strength and appearance:

| SPI Range | Stitch Length | Application | Strength | Appearance |
|-----------|--------------|-------------|----------|------------|
| 5–6 SPI | 4.2–5.0 mm | Overlocking, edge finishing | Low | Loose, visible |
| 7–8 SPI | 3.2–3.6 mm | Standard seams, side construction | Moderate | Balanced |
| 9–10 SPI | 2.5–2.8 mm | Handles, reinforced areas | High | Dense, strong |
| 11–12 SPI | 2.1–2.3 mm | Heavy-duty handles, industrial | Very high | Very dense |

### SPI Selection Guidelines

**Side seams (bag body)**:
- Standard load: 7–8 SPI
- Heavy load: 8–9 SPI

**Handle attachments**:
- Basic loop: 9–10 SPI
- Reinforced (X-stitch, box-stitch): 10–12 SPI
- Industrial/heavy duty: 12+ SPI with bonded nylon thread

**Bottom seams**:
- Flat bottom: 7–8 SPI
- Box bottom: 8–9 SPI (higher stress at corners)

**Quality rule**: Never go below 7 SPI for load-bearing seams. Low SPI creates long stitch lengths that propagate tears easily.

## Seam Types

### Overlock, Chain Stitch, Lock Stitch

Different seam types serve different structural and aesthetic purposes:

### Overlock (Serger) Stitch
- **Stitch class**: 504, 514 (3-thread or 4-thread)
- **Function**: Edge finishing to prevent fraying
- **Characteristics**: Wraps thread around fabric edge; moderate strength
- **Best for**: Neatening raw edges on unlaminated nonwoven
- **Limitation**: Not suitable as primary structural seam for loaded bags

### Chain Stitch
- **Stitch class**: 401 (single needle), 404 (zigzag variant)
- **Function**: Fast, stretchable seam for bag body construction
- **Characteristics**: Looped chain formation; easy to unravel if broken
- **Best for**: Side seams on light-to-medium duty bags
- **Speed advantage**: 20–30% faster than lock stitch

### Lock Stitch
- **Stitch class**: 301 (straight lock), 304 (zigzag lock)
- **Function**: Maximum security seam
- **Characteristics**: Needle thread and bobbin thread interlock; does not unravel
- **Best for**: Handles, reinforced areas, premium bags
- **Strength**: Highest seam security; if one stitch breaks, seam stays intact

### Seam Selection Matrix

| Bag Type | Primary Seam | Edge Finish | Handle Attachment |
|----------|-------------|-------------|-------------------|
| Light giveaway | Chain stitch (401) | Overlock | Chain stitch |
| Standard retail | Chain stitch (401) | Overlock or fold | Lock stitch (301) |
| Premium gift | Lock stitch (301) | Folded edge | Lock stitch + bar-tack |
| Heavy duty | Lock stitch (301) | Folded edge | Lock stitch + X-stitch |
| Industrial | Lock stitch (301) | Folded + bound | Lock stitch + box-stitch |

## Handle Attachment Methods

### Box X, Straight Bar, Cross Stitch

Handle attachment is the highest-stress zone in any bag. Method selection determines failure mode and load capacity:

### Straight Bar Stitch
- **Pattern**: Single or double row of straight stitches across handle base
- **Stitches**: 2–4 rows, 7–9 SPI
- **Strength**: Basic; relies on thread tension and fabric integrity
- **Cost**: Lowest (+$0.01–0.02)
- **Use for**: Light loads (< 5 kg), promotional giveaways

### Box Stitch (Rectangular)
- **Pattern**: Rectangular perimeter stitching around handle base
- **Stitches**: Perimeter at 9–10 SPI, optionally filled with diagonal cross
- **Strength**: Good; distributes load around perimeter
- **Cost**: Moderate (+$0.03–0.05)
- **Use for**: Medium loads (5–10 kg), retail shopping bags

### X-Stitch (Cross Stitch)
- **Pattern**: Diagonal lines forming an X over handle base
- **Stitches**: Diagonals at 10–12 SPI, often combined with box perimeter
- **Strength**: Very good; distributes load diagonally across panel
- **Cost**: Moderate (+$0.04–0.06)
- **Use for**: Medium-to-heavy loads (8–15 kg), grocery bags

### Box X-Stitch
- **Pattern**: Box perimeter + diagonal X fill
- **Stitches**: Full perimeter and diagonals at 10–12 SPI
- **Strength**: Maximum; combines perimeter and diagonal distribution
- **Cost**: Highest (+$0.05–0.08)
- **Use for**: Heavy loads (12–20 kg), wine bags, industrial use

| Method | Relative Strength | Cost Adder | Best For |
|--------|-------------------|-----------|----------|
| Straight bar | 1.0× | Baseline | Light promotional |
| Box stitch | 1.5× | +$0.03 | Standard retail |
| X-stitch | 2.0× | +$0.04 | Heavy retail |
| Box X-stitch | 2.5× | +$0.06 | Maximum durability |

## Quality Checkpoints

### Skip Stitch, Thread Breakage, Tension

Systematic inspection at key production stages prevents defective bags from reaching customers:

### Pre-Production Checkpoints

| Check | Frequency | Acceptable Standard |
|-------|-----------|---------------------|
| Thread tension | Every machine start | Upper: 150–200 g; Lower: 100–150 g |
| Needle condition | Every 500 bags | No burrs, correct size, proper point |
| Stitch length calibration | Every 1,000 bags | Within ±0.5 mm of target SPI |

### In-Production Checkpoints

| Defect | Description | Reject Threshold |
|--------|-------------|------------------|
| Skip stitch | Missing stitches in seam | > 1 skip per 10 cm |
| Thread breakage | Broken thread leaving gap | Any visible breakage |
| Uneven tension | Loose or puckered stitches | Visible puckering or loops |
| Needle holes | Excessive fabric damage | Holes > 1 mm diameter |
| Misalignment | Seam not following pattern | > 2 mm deviation |

### Finished Bag Inspection

| Checkpoint | Method | Standard |
|-----------|--------|----------|
| Seam strength | Pull test (10 N for 5 sec) | No seam opening, no thread breakage |
| Handle pull-out | Static load (2× rated capacity) | No handle separation |
| Stitch count | Visual count per 5 cm | Within ±1 SPI of specification |
| Thread ends | Visual inspection | Trimmed to < 3 mm; no loose ends |

### AQL Sampling
For bulk orders, apply AQL 2.5 sampling (see Article #32 for detailed AQL guidance):

- **Major defects** (seam failure, handle separation): AQL 2.5
- **Minor defects** (uneven stitches, untrimmed threads): AQL 4.0
