# SKILL-00B + SKILL-18: 竞对调研与内容差距分析

**用途**: 分析竞争对手内容，找出差距和机会
**执行时机**: 关键词调研（`keyword-research.md`）完成后

---

## 1. 竞对识别

**直接竞争对手**:
- 同行业制造商/供应商网站
- 行业权威站点/博客
- 产品评测专业站点

**间接竞争对手**:
- 行业论坛/社区
- YouTube 技术频道/评测视频
- Reddit 相关板块
- 行业咨询公司技术文档

---

## 2. 竞对分析维度

**内容结构分析**:
```
□ 文章长度（字数）
□ 章节数量
□ 图片/图表数量
□ 表格/数据表使用情况
□ 视频嵌入
□ FAQ 章节
□ CTA 位置和形式
```

**SEO 元素分析**:
```
□ Title 格式和长度
□ Meta Description
□ H1/H2/H3 结构
□ 内部链接数量
□ 外部链接来源（权威站点/标准组织）
□ 关键词密度
□ Schema 标记使用
```

**专业深度分析**:
```
□ 是否提供详细规格数据
□ 是否引用行业标准/测试数据
□ 是否包含计算工具/图表
□ 是否提供可下载资源（PDF/CAD）
□ 是否包含案例研究/实际应用
```

**用户体验分析**:
```
□ 页面加载速度
□ 移动端适配
□ 技术文档可读性
□ 导航清晰度
□ 询盘/联系入口
```

## 4. 内容差距分析（SKILL-18）

**找出对手未覆盖的主题**:

```
方法:
1. 列出前 5 名竞争对手
2. 提取每篇文章的小标题
3. 找出共同缺失的主题
4. 确定我们可以补充的内容

示例发现:
- 对手 A: 缺少 [主题 A]
- 对手 B: 没有 [主题 B]
- 对手 C: 缺少 [视角 C]
- 对手 D: 无 [内容类型 D]
- 对手 E: 缺少最新 [产品/主题]

→ 策略：覆盖所有缺失点，创建更全面的内容
```

## 5. 反向链接机会

**分析对手外链来源**:

```
工具：Ahrefs / Semrush / Moz

查找:
- 对手获得外链的页面
- 外链类型（行业目录、协会资源、技术论坛）
- 可复制的外链策略

高价值外链来源:
- 行业协会资源页
- 工程参考站点
- 行业目录/数据库
- 技术论坛推荐
- 学术/研究机构引用

我们的行动:
1. 创建原创数据/图表供他人引用
2. 发布行业标准解读系列
3. 制作可下载的资源（手册/计算器）
4. 在行业论坛提供专业解答
```

## 6. 差异化策略

**B2B 专注**（大多数竞争对手聚焦终端用户）:
- 强调批发/OEM 服务能力
- 添加 MOQ、价格区间、交货期信息
- 提供样品请求流程
- 展示工厂/生产/质检能力
- 包含项目案例研究

**数据驱动**（原创内容）:
- 原创测试数据/图表
- 材料性能对比表
- 行业趋势报告
- 失效分析案例
- 客户案例研究

**专业深度**（超越对手）:
- 完整的标准引用（ASME/API/ASTM 等）
- 详细的规格参数表
- 焊接/安装工艺指导
- 检测要求说明
- 更多实际应用案例

**采购支持**:
- 在线询价系统
- 技术规格书下载
- CAD 图纸库
- 材料证书样本
- 质检流程说明
