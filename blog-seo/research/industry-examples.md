# 行业特定关键词与竞对参考

本文档包含不同行业的特定关键词策略和竞争对手参考，供 `keyword-research.md` 和 `competitor-research.md` 使用时参考。

---

## 刀具行业 (Knife Industry)

### B2B 导向关键词
```
- wholesale knives
- knife manufacturer
- OEM knife service
- private label knives
- custom knife maker
- Yangjiang knives
```

### 产品评测关键词
```
- [钢材名] steel review
- is [钢材名] good for knives
- [钢材 A] vs [钢材 B]
- best steel for [用途]
- where are [品牌] knives made
```

### 直接竞争对手
```
- bladehq.com/articles
- knifesteelnerds.com
- zknives.com
- bestknifesteel.org
- knifecenter.com/blog
```

### 间接竞争对手
```
- BladeForums 论坛
- r/knives Reddit
- YouTube 刀具频道
```

### 差异化策略
```
B2B 专注:
- 强调批发/OEM 服务
- 添加 MOQ、价格区间信息
- 提供样品请求流程
- 展示工厂/生产能力
- Yangjiang 产业优势

数据驱动:
- 钢材性能测试数据（引用 Knife Steel Nerds）
- CATRA 测试数据
- 硬度 HRC 范围
- 化学成分百分比
```

## 工业品行业 (Flanges/Valves/Pipe Fittings)

### B2B 导向关键词
```
- wholesale pipe fittings
- flange manufacturers
- industrial valve suppliers
- OEM pipe fittings
- ASME certified flanges
- API 6D valve manufacturers
- bulk pipe fittings
```

### 产品评测关键词
```
- [产品名] review
- [材质] flange specifications
- [阀门类型] valve working principle
- [产品 A] vs [产品 B]
- best [产品] for [应用]
```

### 指南类关键词
```
- how to select flange class
- how to install [产品]
- guide to [产品类别]
- types of [类别]
- what is [标准号]
- flange bolt torque chart
- valve pressure temperature rating
```

### 标准相关关键词
```
- ASME B16.5 flanges
- ASME B16.9 fittings
- API 6D valves
- ASTM A105 specifications
- DIN flange standards
- JIS flange dimensions
```

### 直接竞争对手
```
- engineeringtoolbox.com
- pipefittingworld.com
- 同行业制造商网站
- 工业管道产品供应商博客
```

### 间接竞争对手
```
- 工程咨询公司技术文档
- 行业协会资源页（ASME, API）
- YouTube 技术频道（安装视频、产品评测）
- 行业论坛（Eng-Tips, CR4）
```

### 差异化策略
```
B2B 专注:
- 强调批发/OEM 服务能力
- 添加 MOQ、交货期、认证信息
- 提供样品请求流程
- 展示工厂/生产/质检能力
- 包含项目案例研究

数据驱动:
- 材料性能测试数据
- 压力 - 温度额定值图表
- 尺寸重量对比表
- 行业趋势报告
- 失效分析案例

技术深度:
- 完整的 ASTM/ASME/API 标准引用
- 详细的材料化学成分表
- 焊接工艺指导
- NDT 检测要求说明
- 更多实际应用案例

采购支持:
- 在线询价系统
- 技术规格书下载
- CAD 图纸库
- 材料证书样本
- 质检流程说明
```
