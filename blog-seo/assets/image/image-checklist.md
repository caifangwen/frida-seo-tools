## 🎯 Alt 文本生成公式

### 基础公式

```
Alt 文本 = [主题] + [描述] + [可选细节]

示例:
- weld-neck-flange + dimensions + chart → weld-neck-flange-dimensions-chart
- knife + sharpening + whetstone → knife-sharpening-whetstone-professional
- steel + corrosion + test → steel-corrosion-resistance-salt-spray-test
```

### 增强公式（推荐）

```
Alt 文本 = [具体对象] + [动作/状态] + [场景/环境] + [可选细节]

示例:
- worker + tig welding + stainless flange pipe + industrial setting → worker-tig-welding-stainless-flange-pipe-industrial
- chef knife + cutting vegetables + professional kitchen + closeup → chef-knife-cutting-vegetables-professional-kitchen-closeup
- hardness tester + measuring + steel sample + rockwell-scale → hardness-tester-measuring-steel-sample-rockwell-scale
```

### Alt 文本长度指南

| 类型 | 长度 | 示例 |
|-----|------|------|
| 简单图片 | 3-5 词 | `weld-neck-flange-dimensions` |
| 场景图片 | 5-8 词 | `worker-welding-flange-pipe-industrial` |
| 对比图表 | 6-10 词 | `class-150-vs-300-flange-pressure-rating-comparison-chart` |
| 产品 Figure | 8-12 词 | `oem-fixed-blade-knife-g10-handle-s30v-blade-kkfb00012` |

## ✅ 关键词选择原则

### 正确做法

```
✅ 使用具体描述性关键词
   - `weld neck flange dimensions` 而非 `flange`
   - `chef knife professional kitchen` 而非 `knife`

✅ 添加材质/场景限定
   - `stainless steel 316 flange`
   - `industrial pipeline oil and gas`

✅ 使用同义词扩展
   - `flange` / `fitting` / `connection`
   - `blade` / `knife` / `cutlery`
   - `pipeline` / `pipe` / `tubing`
```

### 错误做法

```
❌ 过于宽泛
   - `metal`, `steel`, `pipe` → 结果不相关

❌ 使用随机种子图片
   - Picsum 随机图

❌ 与主题无关的通用图片
   - `office desk`, `person working`
```
