# M390 vs S30V vs S35VN 钢材对比

| **Edge Retention**       | 9/10        | 7/10           | 6/10        |
| **Toughness**            | 4/10        | 5/10           | 4/10        |
| **Corrosion Resistance** | 9/10        | 7/10           | 7/10        |
| **Wear Resistance**      | 9/10        | 8/10           | 7/10        |
| **Sharpening Ease**      | 5/10        | 7/10           | 8/10        |
| **Price Range**          | High        | Moderate       | Moderate    |
| **Best For**             | Premium EDC | All-around use | Tough tasks |

## Summary & Recommendations

| Use Case                  | Recommended Steel                                   |
| ------------------------- | --------------------------------------------------- |
| **Premium EDC**           | M390 - Best edge retention and corrosion resistance |
| **All-around Use**        | S30V - Balanced performance across all categories   |
| **Tough Tasks**           | S35VN - Good toughness, easier to sharpen           |
| **Budget Conscious**      | S35VN - Similar to S30V at lower cost               |
| **Saltwater Environment** | M390 - Superior corrosion resistance                |
